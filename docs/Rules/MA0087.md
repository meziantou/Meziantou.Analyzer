# MA0087 - Parameters with \[DefaultParameterValue\] attributes should also be marked \[Optional\]
<!-- sources -->
Source: [OptionalParametersAttributeAnalyzer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer/Rules/OptionalParametersAttributeAnalyzer.cs)
<!-- sources -->

````csharp
using System.Runtime.InteropServices;
class Test
{
    void A([Optional, DefaultParameterValue(10)]int a)
    {
    }

    void B([DefaultParameterValue(10)]int a) // Non-compliant
    {
    }
}
````
