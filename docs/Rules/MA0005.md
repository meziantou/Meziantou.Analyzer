# MA0005 - Use Array.Empty\<T\>()
<!-- sources -->
Sources: [UseArrayEmptyAnalyzer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer/Rules/UseArrayEmptyAnalyzer.cs), [UseArrayEmptyFixer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer.CodeFixers/Rules/UseArrayEmptyFixer.cs)
<!-- sources -->

You should use `Array.Empty<T>()` instead of `new T[0]` to avoid an allocation.

````csharp
new int[0];

// Should be
Array.Empty<int>();
````
