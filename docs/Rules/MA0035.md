# MA0035 - Do not use dangerous threading methods
<!-- sources -->
Source: [DontUseDangerousThreadingMethodsAnalyzer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer/Rules/DontUseDangerousThreadingMethodsAnalyzer.cs)
<!-- sources -->

This rule warns about using the methods `Abort()`, `Suspend()` or `Resume()` declared by the `Thread` class.
