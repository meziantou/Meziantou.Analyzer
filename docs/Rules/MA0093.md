# MA0093 - EventArgs should not be null
<!-- sources -->
Sources: [EventsShouldHaveProperArgumentsAnalyzer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer/Rules/EventsShouldHaveProperArgumentsAnalyzer.cs), [UseEventArgsEmptyFixer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer.CodeFixers/Rules/UseEventArgsEmptyFixer.cs)
<!-- sources -->

````c#
using System;
class Test
{
    public event EventHandler MyEvent;

    void OnEvent()
    {
        MyEvent.Invoke(this, null); // non-compliant
        MyEvent.Invoke(this, EventArgs.Empty); // ok
    }
}
````
