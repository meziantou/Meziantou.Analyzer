# MA0093 - EventArgs should not be null when raising an event
<!-- sources -->
Sources: [EventsShouldHaveProperArgumentsAnalyzer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer/Rules/EventsShouldHaveProperArgumentsAnalyzer.cs), [UseEventArgsEmptyFixer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer.CodeFixers/Rules/UseEventArgsEmptyFixer.cs)
<!-- sources -->

This rule detects when an event is raised (invoked) with a `null` EventArgs argument. The rule specifically checks event invocations (calling `.Invoke()` on an event delegate) and does not apply to regular method calls that accept EventArgs parameters.

````c#
using System;
class Test
{
    public event EventHandler MyEvent;

    void OnEvent()
    {
        MyEvent.Invoke(this, null); // non-compliant: raising event with null EventArgs
        MyEvent.Invoke(this, EventArgs.Empty); // compliant: using EventArgs.Empty

        // Note: This rule only applies to event invocations, not regular method calls
        RegularMethod(null); // not flagged by this rule
    }

    void RegularMethod(EventArgs args) { }
}
````
