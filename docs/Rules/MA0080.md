# MA0080 - Use a cancellation token using .WithCancellation()
<!-- sources -->
Source: [UseAnOverloadThatHasCancellationTokenAnalyzer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer/Rules/UseAnOverloadThatHasCancellationTokenAnalyzer.cs)
<!-- sources -->

````c#
IAsyncEnumeration<string> enumerable;
await foreach (var item in enumerable) // not compliant
{
}

await foreach (var item in enumerable.WithCancellation(cancellationToken)) // compliant
{
}
````
