# MA0156 - Use 'Async' suffix when a method returns IAsyncEnumerable\<T\>
<!-- sources -->
Source: [MethodsReturningAnAwaitableTypeMustHaveTheAsyncSuffixAnalyzer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer/Rules/MethodsReturningAnAwaitableTypeMustHaveTheAsyncSuffixAnalyzer.cs)
<!-- sources -->

Methods that return `IAsyncEnumerable<T>` should have the Async suffix.

````c#
// compliant
IAsyncEnumerable<string> FooAsync() => throw null;

// non-compliant
IAsyncEnumerable<string> Foo() => throw null;
````
