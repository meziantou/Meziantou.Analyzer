# MA0049 - Type name should not match containing namespace
<!-- sources -->
Source: [TypeNameMustNotMatchNamespaceAnalyzer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer/Rules/TypeNameMustNotMatchNamespaceAnalyzer.cs)
<!-- sources -->

For usability, you should not give a type the same name as its containing namespace, as this makes the type harder to use in code.

````csharp
namespace Foo
{
    class Foo // non compliant
    {
    }
}

var foo = new Foo.Foo(); // You must fully qualify the type
````
