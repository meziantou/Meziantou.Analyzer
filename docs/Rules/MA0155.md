# MA0155 - Do not use async void methods
<!-- sources -->
Source: [DoNotUseAsyncVoidAnalyzer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer/Rules/DoNotUseAsyncVoidAnalyzer.cs)
<!-- sources -->

```c#
// not-compliant
async void SomeMethod()
{
    await Task.Delay(1000);
}

// ok
async Task SomeMethod()
{
    await Task.Delay(1000);
}
```
