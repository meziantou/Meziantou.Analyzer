# MA0091 - Sender should be 'this' for instance events
<!-- sources -->
Sources: [EventsShouldHaveProperArgumentsAnalyzer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer/Rules/EventsShouldHaveProperArgumentsAnalyzer.cs), [EventsShouldHaveProperArgumentsFixer.MA0091.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer.CodeFixers/Rules/EventsShouldHaveProperArgumentsFixer.MA0091.cs)
<!-- sources -->

````c#
using System;
class Test
{
    public event EventHandler MyEvent;

    void OnEvent()
    {
        MyEvent.Invoke(null, EventArgs.Empty); // non-compliant
        MyEvent.Invoke(this, EventArgs.Empty); // ok
    }
}
````
