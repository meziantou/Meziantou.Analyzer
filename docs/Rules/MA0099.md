# MA0099 - Use Explicit enum value instead of 0
<!-- sources -->
Source: [DoNotUseZeroToInitializeAnEnumValue.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer/Rules/DoNotUseZeroToInitializeAnEnumValue.cs)
<!-- sources -->

````c#
enum MyEnum { A = 0, B = 1 }

class Test
{
    void A()
    {
        MyEnum a = 0; // Report diagnostic
        MyEnum b = (MyEnum)0;
        MyEnum c = MyEnum.A;
        MyEnum d = MyEnum.B;
        long e = 0;
        long f = (long)0;
    }
}
````
