# MA0064 - Avoid locking on publicly accessible instance
<!-- sources -->
Source: [AvoidLockingOnPubliclyAccessibleInstanceAnalyzer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer/Rules/AvoidLockingOnPubliclyAccessibleInstanceAnalyzer.cs)
<!-- sources -->

````csharp
lock(this) { } // non compliant
````

````csharp
lock(typeof(Console)) { } // non compliant

var type = obj.GetType();
lock (type) { } // non compliant
````

````csharp
public class Test
{
    public object PublicField;
    public void A()
    {
        lock (PublicField) { } // non compliant
    }
}
````
