# MA0058 - Class name should end with 'Exception'
<!-- sources -->
Source: [ExceptionNameShouldEndWithExceptionAnalyzer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer/Rules/ExceptionNameShouldEndWithExceptionAnalyzer.cs)
<!-- sources -->

A class that derives from `System.Exception` should have a name that ends with 'Exception'.

````csharp
class Sample : Exception // Non compliant
{
}
````

````csharp
class SampleException : Exception // compliant
{
}
````
