# MA0147 - Avoid async void method for delegate
<!-- sources -->
Source: [DoNotUseAsyncDelegateForSyncDelegateAnalyzer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer/Rules/DoNotUseAsyncDelegateForSyncDelegateAnalyzer.cs)
<!-- sources -->

````c#
Foo(() => {}); // ok

Foo(async () => {}); // non-compliant as the delegate is not expecting an async method

void Foo(System.Action action) => throw null;
````
