# MA0077 - A class that provides Equals(T) should implement IEquatable<T>

## Non-compliant code

```csharp
using System;
class MyClass
{
    public bool Equals(MyClass other)
    {
        ...
    }
}
```

## Compliant code

```csharp
using System;
class MyClass : IEquatable<MyClass>
{
    public bool Equals(MyClass other)
    {
        ...
    }
}
```
