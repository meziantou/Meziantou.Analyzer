# MA0052 - Replace constant Enum.ToString with nameof
<!-- sources -->
Sources: [ReplaceEnumToStringWithNameofAnalyzer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer/Rules/ReplaceEnumToStringWithNameofAnalyzer.cs), [ReplaceEnumToStringWithNameofFixer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer.CodeFixers/Rules/ReplaceEnumToStringWithNameofFixer.cs)
<!-- sources -->

You should use `nameof` instead of calling `ToString` on a constant enumeration value. This is mainly for performance reason.

````csharp
public enum Color
{
    Blue,
    Red,
    Yellow,
}

void Sample()
{
    _ = Color.Blue.ToString(); // non compliant
    _ = nameof(Color.Blue); // compliant
}
````
