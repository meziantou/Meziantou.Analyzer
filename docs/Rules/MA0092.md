# MA0092 - Sender should be 'null' for static events
<!-- sources -->
Source: [EventsShouldHaveProperArgumentsAnalyzer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer/Rules/EventsShouldHaveProperArgumentsAnalyzer.cs)
<!-- sources -->

````c#
using System;
class Test
{
    public static event EventHandler MyEvent;

    void OnEvent()
    {
        MyEvent.Invoke(this, EventArgs.Empty); // non-compliant
        MyEvent.Invoke(null, EventArgs.Empty); // ok
    }
}
````
