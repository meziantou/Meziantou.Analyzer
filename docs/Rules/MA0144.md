# MA0144 - Use System.OperatingSystem to check the current OS
<!-- sources -->
Sources: [UseOperatingSystemInsteadOfRuntimeInformationAnalyzer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer/Rules/UseOperatingSystemInsteadOfRuntimeInformationAnalyzer.cs), [UseOperatingSystemInsteadOfRuntimeInformationFixer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer.CodeFixers/Rules/UseOperatingSystemInsteadOfRuntimeInformationFixer.cs)
<!-- sources -->

Use `System.OperatingSystem` to check the current OS instead of `RuntimeInformation`.

````c#
RuntimeInformation.IsOSPlatform(OSPlatform.Windows); // non-compliant

OperatingSystem.IsWindows(); // ok
````
