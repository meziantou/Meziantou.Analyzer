# MA0012 - Do not raise reserved exception type
<!-- sources -->
Source: [DoNotRaiseReservedExceptionTypeAnalyzer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer/Rules/DoNotRaiseReservedExceptionTypeAnalyzer.cs)
<!-- sources -->

The following exception types are reserved and should be thrown only by the Common Language Runtime:

- `System.ExecutionEngineException`
- `System.IndexOutOfRangeException`
- `System.NullReferenceException`
- `System.OutOfMemoryException`
- `System.StackOverflowException`
- `System.InvalidProgramException`
- `System.AccessViolationException`
- `System.CannotUnloadAppDomainException`
- `System.BadImageFormatException`
- `System.DataMisalignedException`
