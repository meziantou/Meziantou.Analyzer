# MA0081 - Method overrides should not omit params keyword
<!-- sources -->
Sources: [PreserveParamsOnOverrideAnalyzer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer/Rules/PreserveParamsOnOverrideAnalyzer.cs), [PreserveParamsOnOverrideFixer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer.CodeFixers/Rules/PreserveParamsOnOverrideFixer.cs)
<!-- sources -->

````csharp
class Base
{
  public virtual void Method(params int[] numbers) => { };
}

class Derived : Base
{
  public override void Method(int[] numbers) // non-compliant: "params" is missing
  {
  }
}
````
