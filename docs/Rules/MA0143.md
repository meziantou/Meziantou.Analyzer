# MA0143 - Primary constructor parameters should be readonly
<!-- sources -->
Source: [PrimaryConstructorParameterShouldBeReadOnlyAnalyzer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer/Rules/PrimaryConstructorParameterShouldBeReadOnlyAnalyzer.cs)
<!-- sources -->

The rule reports all assignments to primary constructor parameters

````c#
class Sample(int p)
{
    void A()
    {
        p = 0; // non-compliant
        (p, _) = (0, 1); // non-compliant
    }
}
````
