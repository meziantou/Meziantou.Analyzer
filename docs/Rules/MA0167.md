# MA0167 - Use an overload with a TimeProvider argument
<!-- sources -->
Source: [UseAnOverloadThatHasTimeProviderAnalyzer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer/Rules/UseAnOverloadThatHasTimeProviderAnalyzer.cs)
<!-- sources -->


You should pass a `System.TimeProvider` when calling a method if there is an overload of the method that supports it.

````csharp
class Test
{
    public void A()
    {
        Task.Delay(default(TimeSpan)); // non-compliant
    }
}
````
