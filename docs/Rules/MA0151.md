# MA0151 - DebuggerDisplay must contain valid members
<!-- sources -->
Source: [DebuggerDisplayAttributeShouldContainValidExpressionsAnalyzer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer/Rules/DebuggerDisplayAttributeShouldContainValidExpressionsAnalyzer.cs)
<!-- sources -->

Validates the expressions within `[DebuggerDisplay]` attributes to ensure they reference valid members.

````c#
[DebuggerDisplay("{Display}")] // compliant
public class Dummy1
{
    public string Display { get; }
}

[DebuggerDisplay("{InvalidMember}")] // non-compliant as InvalidMember doesn't exist
public class Dummy2
{
    public string Display { get; }
}
````
