# MA0006 - Use String.Equals instead of equality operator
<!-- sources -->
Sources: [UseStringEqualsAnalyzer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer/Rules/UseStringEqualsAnalyzer.cs), [UseStringEqualsFixer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer.CodeFixers/Rules/UseStringEqualsFixer.cs)
<!-- sources -->

You should use `string.Equals` instead of `==` or `!=`, to make string comparison rules explicit.

````csharp
"" == "";

// Should be
string.Equals("", "", StringComparison.Ordinal);
````

````csharp
"" != "";

// Should be
!string.Equals("", "", StringComparison.Ordinal);
````
