# MA0171 - Use pattern matching instead of HasValue for Nullable\<T\> check
<!-- sources -->
Sources: [UsePatternMatchingInsteadOfHasValueAnalyzer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer/Rules/UsePatternMatchingInsteadOfHasValueAnalyzer.cs), [UsePatternMatchingInsteadOfHasvalueFixer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer.CodeFixers/Rules/UsePatternMatchingInsteadOfHasvalueFixer.cs)
<!-- sources -->

Use pattern matching instead of the `HasValue` property to check nullable value types.

````c#
int? value = null;

_ = value.HasValue;    // non-compliant
_ = value is not null; // compliant
````
