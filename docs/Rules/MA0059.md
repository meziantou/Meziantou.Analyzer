# MA0059 - Class name should end with 'EventArgs'
<!-- sources -->
Sources: [EventArgsNameShouldEndWithEventArgsAnalyzer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer/Rules/EventArgsNameShouldEndWithEventArgsAnalyzer.cs), [TypeNameShouldEndWithSuffixFixer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer.CodeFixers/Rules/TypeNameShouldEndWithSuffixFixer.cs)
<!-- sources -->

A class that derives from `System.EventArgs` should have a name that ends with 'EventArgs'.

````csharp
class Sample : EventArgs // Non compliant
{
}
````

````csharp
class SampleEventArgs : EventArgs // compliant
{
}
````
