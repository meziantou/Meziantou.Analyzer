# MA0131 - ArgumentNullException.ThrowIfNull should not be used with non-nullable types
<!-- sources -->
Sources: [ThrowIfNullWithNonNullableInstanceAnalyzer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer/Rules/ThrowIfNullWithNonNullableInstanceAnalyzer.cs), [ThrowIfNullWithNonNullableInstanceFixer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer.CodeFixers/Rules/ThrowIfNullWithNonNullableInstanceFixer.cs)
<!-- sources -->

`ArgumentNullException.ThrowIfNull` should not be used with non-nullable value, such as `int` or `bool`.

````c#
void Sample(int value, string str)
{
    ArgumentNullException.ThrowIfNull(value); // non-compliant as int is not nullable
    ArgumentNullException.ThrowIfNull(str); // ok as string is a reference type
}
````
