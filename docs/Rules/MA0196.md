# MA0196 - Do not use inheritdoc on non-inheriting members
<!-- sources -->
Source: [InheritdocShouldBeUsedOnInheritingMemberAnalyzer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer/Rules/InheritdocShouldBeUsedOnInheritingMemberAnalyzer.cs)
<!-- sources -->

Use `<inheritdoc />` only when documentation is inherited from a base member implementation.

This rule reports `<inheritdoc />` when the member is not:
- an override, and
- an interface implementation.

`<inheritdoc .../>` with a `cref` attribute is allowed.

````csharp
// Non-compliant
class Sample
{
    /// <inheritdoc />
    public void M() { }
}

// Compliant (override)
class BaseType
{
    public virtual void M() { }
}

class DerivedType : BaseType
{
    /// <inheritdoc />
    public override void M() { }
}

// Compliant (explicit cref)
class OtherSample
{
    /// <inheritdoc cref="object.ToString" />
    public void M() { }
}
````
