# MA0044 - Remove useless ToString call
<!-- sources -->
Sources: [RemoveUselessToStringAnalyzer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer/Rules/RemoveUselessToStringAnalyzer.cs), [RemoveUselessToStringFixer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer.CodeFixers/Rules/RemoveUselessToStringFixer.cs)
<!-- sources -->

`string.ToString` is a no-op call. You should remove the call to `ToString()`.

````csharp
var str = "abc";
Console.Write(str.ToString()); // non compliant
````

````csharp
var str = "abc";
Console.Write(str); // compliant
````
