# MA0168 - Use readonly struct for in or ref readonly parameter
<!-- sources -->
Source: [UseReadOnlyStructForRefReadOnlyParametersAnalyzer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer/Rules/UseReadOnlyStructForRefReadOnlyParametersAnalyzer.cs)
<!-- sources -->

```c#
void A(in Foo p) { } // not-compliant as Foo is not readonly

struct Foo { }
```

```c#
void A(in Foo p) { } // ok

readonly struct Foo { }
```
