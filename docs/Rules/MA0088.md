# MA0088 - Use \[DefaultParameterValue\] instead of \[DefaultValue\]
<!-- sources -->
Sources: [OptionalParametersAttributeAnalyzer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer/Rules/OptionalParametersAttributeAnalyzer.cs), [OptionalParametersAttributeFixer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer.CodeFixers/Rules/OptionalParametersAttributeFixer.cs)
<!-- sources -->

````csharp
using System.ComponentModel;
using System.Runtime.InteropServices;

class Test
{
    void A([DefaultValue(10)]int a) { } // non-compliant
    void A([DefaultParameterValue(10)]int a) { } // ok
    void A([DefaultValue(10), DefaultParameterValue(10)]int a) { } // ok
}
````
