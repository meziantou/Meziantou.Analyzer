# MA0130 - GetType() should not be used on System.Type instances
<!-- sources -->
Source: [ObjectGetTypeOnTypeInstanceAnalyzer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer/Rules/ObjectGetTypeOnTypeInstanceAnalyzer.cs)
<!-- sources -->

`GetType()` returns the current type. Most of the time, when used on an instance of type `System.Type`, it's an error.

````c#
System.Type type = ...;
type.GetType(); // non-compliant

int a = 123;
a.GetType(); // ok
````
