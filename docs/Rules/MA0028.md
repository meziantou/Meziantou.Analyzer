# MA0028 - Optimize StringBuilder usage

Sources: [OptimizeStringBuilderUsageAnalyzer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer/Rules/OptimizeStringBuilderUsageAnalyzer.cs), [OptimizeStringBuilderUsageFixer.cs](https://github.com/meziantou/Meziantou.Analyzer/blob/main/src/Meziantou.Analyzer.CodeFixers/Rules/OptimizeStringBuilderUsageFixer.cs)
<!-- sources -->

```csharp
new StringBuilder().Append($"a{10}");
new StringBuilder().Append("a" + 10);

// Should be
new StringBuilder().Append("a").Append(10);
```

```csharp
new StringBuilder().Append("a");

// Should be
new StringBuilder().Append('a');
```

```csharp
new StringBuilder().Append(10.ToString());

// Should be
new StringBuilder().Append(10);
```
